# Semana 4 – Regularización en Redes Neuronales

## Objetivo
Aplicar técnicas de regularización para reducir el overfitting en una red neuronal.

## Métodos aplicados
Se utilizaron Dropout y regularización L2.

## Comparación
Se comparó un modelo base sin regularización vs uno con regularización usando el mismo dataset y arquitectura.

## Resultado principal
El modelo con regularización mostró mejor generalización al reducir la diferencia entre entrenamiento y validación.
