# Regularización en Redes Neuronales

## Objetivo
Aplicar técnicas de regularización para reducir el overfitting en una red neuronal.

## Métodos aplicados
Se utilizó Dropout y regularización L2.

## Comparación
Se comparó un modelo base sin regularización vs uno con regularización usando el mismo dataset y arquitectura.

## Resultado principal
El modelo regularizado mostró mejor generalización y menor overfitting.

Abrir el notebook en Google Colab y ejecutar todas las celdas.
